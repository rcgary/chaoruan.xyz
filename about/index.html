关于我

坐标墨尔本，做了十几年软件开发，写代码养家糊口。家里有一只叫 Cinnamon 的猫和一只叫 Leo 的狗。

游戏是第一爱好，什么都玩，从独立小品到3A大作，只要好玩就行。不挑类型，不挑平台，PC 为主，主机也玩。一旦上头可以连续打好几天，打完了又开始找下一个。偶尔也去健身房，主要是为了心安理得地继续坐着打游戏。

喜欢折腾各种乱七八糟的东西，效率工具、自动化、智能家居，能自动的绝不手动。折腾的时间可能比实际省下的时间还多，但这不是重点。

博客写了好几年，断断续续的，写的远比看的少。内容随缘，技术、游戏、旅行、吃的、生活碎碎念，什么都可能出现。英文版也有，但更新频率基本靠缘分。

工具

  • 编辑器: Neovim
  • 笔记: Obsidian
  • 终端: WezTerm + tmux
  • 常用命令行: fzf, ripgrep, bat, lazygit, yazi, zoxide
  • 主题: Catppuccin Mocha（全家桶）
  • 字体: Fira Code
  • 配置管理: chezmoi（一套配置跑所有机器）
