### Install Setup

Install [hugo](https://gohugo.io/installation/)
``` sh
brew install hugo
```

Install theme - [LoveIt](https://github.com/dillonzq/LoveIt)

``` sh
cd theme
git clone git@github.com:dillonzq/LoveIt.git
```

Run the blog locally

``` sh
# Under the Root
hugo server -D --navigateToChanged
```
